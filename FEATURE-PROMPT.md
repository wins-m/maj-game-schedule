# 功能需求提示词

为麻将比赛管理系统添加牌谱网址和评分记录功能：

## 核心需求

1. **积分录入增强**：在现有积分录入对话框中，按桌分组显示，每桌4人可同时录入：
   - 积分（已有）
   - 评分（新增，字符串类型）
   - 牌谱网址（新增，每桌一个URL）

2. **历史记录页面**：新增子页面（如 `history.html`），表格展示：
   - 场次、桌次
   - 4人姓名、积分、评分
   - 牌谱网址（超链接）
   - 支持编辑和补录

3. **数据模型扩展**：
   - Game对象中每桌增加 `scores`（积分对象）、`ratings`（评分对象）、`recordUrl`（牌谱网址）
   - 保持向后兼容，历史数据自动补全默认值

4. **后端API**：无需新增接口，使用现有 `/api/games` 保存扩展后的数据
